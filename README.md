# permissions-center

## 介绍

功能丰富且强大后台系统

- 主要语言: Java

- 数据库: MySQl

- NoSQL: Redis

- 技术栈: `SpringBoot`, `MyBatis`, `Netty`, `RocketMQ`, `KafKa`, `ShardingJdbc`, `ElasticsSearch`

核心功能:

1. 完整的权限管理
2. 动态线程池监控
3. ShardingJdbc动态切换数据源, 分库分表
4. Netty的即时通讯
5. ES数据检索
6. JavaCV视频监控
7. MySQL binlog数据订阅
8. 多方式文件上传下载
9. 大数据excel导入导出
